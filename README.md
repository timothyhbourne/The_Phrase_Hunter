# The_Phrase_Hunter
 A game app I created similar to the game of Hangman. This project is part of my unit 4 journey in Treehouse' tech degree program.

# Timer
Added a timer to make the game more interesting. If timer reaches 0 before user guesses the word, the game is lost.

# CSS
Added modifications to button hover state. The color will change if the mouse hovers the on screen keyboard.
